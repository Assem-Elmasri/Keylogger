# Keylogger

> simple keylogger built in c for **educational purposes**

## Installation

```bash
gcc -Wall -Wextra -g -o build/keylogger keylogger.c
```

this will lead to compiling the project in the build file within the repo

## Usage

```bash
./build/keylogger <event-file>
```

this project works on linux specifaclly the `/dev/input/eventN` file for the keyboard
